# 正體中文去 AI 八股文規則

8 條針對正體中文（台灣用語）的去 AI 規則，每條含 before/after 範例。

---

## 1. 華麗起手式

刪除空泛的開場白，直接切入主題。

**常見句型：** 「在當今瞬息萬變的數位時代...」、「隨著科技的蓬勃發展...」、「在數位轉型的浪潮中...」、「在這個資訊爆炸的年代...」

**Before:**
> 在當今瞬息萬變的數位時代，人工智慧正以前所未有的速度改變我們的工作模式，為企業帶來全新的機遇與挑戰。

**After:**
> 過去兩年，多數台灣企業開始在客服和資料分析中導入 AI 工具，內部流程效率平均提升了 20%。

---

## 2. AI 轉折詞替換

避免機械化地使用過度正式的連接詞。

| AI 常用 | 替換為 |
|:---|:---|
| 值得注意的是 | 不過 / 另外 / 這裡要提的是 |
| 不可否認的是 | 確實 / 的確 |
| 總而言之 | 整體來看 / 簡單說 |
| 此外 | 另外 / 還有 |
| 綜上所述 | 回頭來看 / 總結一下 |
| 然而值得關注的是 | 但是 |
| 與此同時 | 同時 / 另一方面 |

**Before:**
> 值得注意的是，此框架不僅提升了開發效率，此外還增強了系統的可維護性。綜上所述，該方案具有廣泛的應用前景。

**After:**
> 這個框架讓開發速度快了不少，維護起來也比較輕鬆。整體來看，適合用在大部分的後端專案。

---

## 3. 大陸/AI 空泛用語 → 台灣平實用語

AI 常產出偏大陸風格或過度包裝的動詞/名詞，替換為台灣業界的日常用語。

| AI/大陸用語 | 台灣慣用語 |
|:---|:---|
| 賦能 | 協助、幫助 |
| 重塑 | 改變、調整 |
| 無縫接軌 | 順利整合、銜接 |
| 提供強大助力 | 有效支援、幫上忙 |
| 展現了...的魅力 | 表現得不錯 |
| 打造 | 建立、做出 |
| 落地 | 實際執行、上線 |
| 抓手 | 具體做法、切入點 |
| 底層邏輯 | 基本原理、核心概念 |
| 生態 / 生態系 | 產業環境 / 相關服務 |
| 範式 | 模式、做法 |
| 閉環 | 完整流程 |
| 鏈路 | 流程、路徑 |
| 頂層設計 | 整體規劃 |
| 沉澱 | 累積、整理 |
| 對齊 | 確認一致、同步 |

**Before:**
> 這套解決方案賦能企業數位轉型，透過底層邏輯的重塑，打造全新的數據生態，實現業務閉環。

**After:**
> 這套方案協助企業做數位轉型，從基本架構開始調整，建立完整的資料處理流程。

---

## 4. 對稱句型打散

AI 偏好整齊的排比句，把它們拆成自然的口語短句。

**常見句型：** 「不僅能...，更能...」、「既...又...」、「一方面...另一方面...」

**Before:**
> 此框架不僅能提升開發效率，更能確保程式碼品質，同時還能降低維運成本，實現開發與運維的無縫接軌。

**After:**
> 這個框架讓開發速度變快，寫出來的程式碼品質也比較穩定。維運那邊的負擔跟著減少，銜接起來順很多。

---

## 5. 三段論空泛結論

AI 喜歡用「總之前景看好」收尾，改為具體的下一步或數據。

**Before:**
> 總而言之，人工智慧的發展前景一片光明，未來必將為人類社會帶來更加深遠的影響，讓我們拭目以待。

**After:**
> 目前團隊預計下季度把 AI 審查功能整合進現有的 CI/CD 流程，預估可以減少約 30% 的人工 code review 時間。

---

## 6. 空泛動詞堆疊

AI 特別愛把「提升」、「優化」、「驅動」、「助力」疊在一起用，每個動詞都很抽象。

**Before:**
> 透過持續優化系統架構，驅動技術創新，助力業務增長，全面提升用戶體驗。

**After:**
> 我們重構了快取層，API 回應時間從 800ms 降到 200ms，使用者反映頁面載入明顯變快。

---

## 7. 過度使用「的」與冗餘修飾

AI 產出的中文常有過多的「的」和不必要的修飾詞。

**Before:**
> 這是一個非常重要的、具有里程碑意義的、值得我們深入思考的技術性的突破。

**After:**
> 這次技術突破很重要，值得花時間研究。

---

## 8. 條列式標題加粗

跟英文的 inline-header lists 一樣，中文 AI 輸出也常用粗體標題 + 冒號的條列格式。

**Before:**
> - **效能提升：** 系統效能獲得了顯著的提升與改善。
> - **成本降低：** 整體營運成本實現了大幅度的削減。
> - **品質改善：** 產品品質達到了更高的標準與水準。

**After:**
> 系統跑得更快了，營運成本跟著壓低，產品品質也拉上來。三個面向都有明顯改善。
