# 兩岸軟體開發用語對照表

處理中文技術文件時，將大陸用語替換為台灣慣用語。

> **注意：請根據上下文語意進行替換，確保技術名詞的準確性，避免過度糾正。** 例如「接口」在硬體語境下指實體連接埠，在軟體語境下才應替換為「介面」；「日誌」在航海或歷史語境下不應替換為「紀錄檔」。
